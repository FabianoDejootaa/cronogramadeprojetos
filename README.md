 # Programa de Cronogramas de Projetos.

  Bem vindo á documentação do projeto de desenvolvimento do Programa de Cronogramas de Projetos!

  Este usuário vem por este, desenvolver um sistema openSource de cronogramas de projetos customizável, onde será possível fazer tudo do jeitinho que você deseja.
  Estou trabalhando em uma base sólida primeiramente para depois fazer algo mais próximo do objetivo deste projeto.

# Funcionalidades

  - Manipulação de usuários e equipes;
  - Manipulação de projetos, codificação e setores;
  - Criação de entradas dentro do projeto como: data de ínicio, data de entrega, data de entrega estimada, tarefas, entre outras;
  - Criação de formulários de desenvolvimento e análise de eficiência e produtividade;
  - Análise de custo e gastos com o projeto;

Projeto esta sendo desenvolvido somente por mim.
